# Axon’s Inventory – Support-Verweise

Diese Datei hält die öffentlichen Support-Ziele fest, die das Modul verwendet.
Seit Phase 8.4 sind sie über den sichtbaren Support-Button im Inventar erreichbar.

- Discord-Kontakt: https://discord.com/users/622739422332321792
- GitHub-Projekt: https://github.com/axon-dystro/BetterInv
- GitHub-Issues: https://github.com/axon-dystro/BetterInv/issues
- Bug melden: https://github.com/axon-dystro/BetterInv/issues/new?title=%5BBug%5D%20
- Feature wünschen: https://github.com/axon-dystro/BetterInv/issues/new?title=%5BFeature%5D%20
- Dokumentation: https://github.com/axon-dystro/BetterInv#readme

Die URLs sind zusätzlich zentral in `scripts/main.js` hinterlegt, damit alle
Support- und Feedback-Schaltflächen dieselben Ziele verwenden. Das Modul sendet
selbst keine Nachrichten, Diagnosedaten oder Formulare an einen eigenen Server.
